# Constancia
Este componente será el que permite a los usuarios descargar un pdf (con la forma de una constancia escolar)
a partir de otro componente de React.

## React-pdf
React pdf es una libreria de react que nos permite habilitar la creación de un pdf y acceder a sus opciones a partir de la estructuración de varios componentes.

## Props
Los props que se le pasan a este componente vienen desde la página de constancia, a la que se accede al momento de poner la asistencia a un evento de una persona.

```js filename="Constancia.tsx"
import React from 'react'
import { Document, Page, Text, Image, View, Font, StyleSheet } from '@react-pdf/renderer'

const Constancia = ({ data } : any) => {

    return (
        <Document>

            <Page size='A4'>
                <View style={
                    {
                        display: 'flex',
                        flexDirection: 'column',
                        justifyContent: 'space-between',
                        height: '100%',
                    }
                }>

                    <Image src='/images/headerConstancia.jpg' style={{
                        objectFit: 'cover',
                    }}></Image>

                    <Text style={{
                        textAlign: 'center',
                    }}>Otorga la presente constancia a:</Text>

                    <Text style={{
                        textAlign: 'center',
                        fontSize: '20px',
                        fontFamily: 'Helvetica-Bold',
                    }}>{data ? data[0] : '...'}</Text>

                    <Text style={{
                        textAlign: 'center',
                    }}>
                        por asistir al evento:
                    </Text>

                    <Text style={{
                        textAlign: 'center',
                        fontSize: '25px',
                        fontFamily: 'Helvetica-Bold',
                        textTransform: 'uppercase',
                    }}>{data ? data[1] : '...'}</Text>

                    <Text style={{
                        textAlign: 'center',
                    }}>Realizado con fecha del día:</Text>

                    <Text style={{
                        textAlign: 'center',
                        textTransform: 'capitalize',
                        fontFamily: 'Helvetica-Bold',
                    }}>{data ? data[2] : '...'}</Text>

                    <Text style={{ textAlign: 'center', fontFamily: 'Helvetica-Bold', marginTop: '40px' }}>{secretario}</Text>
                    <Text style={{ textAlign: 'center', marginTop: '-50px', fontSize: '10px' }}>Secretario Politécnico "Ing. Jorge Matute Remus"</Text>


                    <Image src='/images/footerConstancia.jpg'></Image>
                </View>
            </Page>
        </Document>
    )
}

export default Constancia
```
