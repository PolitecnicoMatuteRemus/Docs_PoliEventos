# Componente de Botones
La función principal de este componente es no cargar tanto código en la página
index. Su composición es de 3 botones que te llevan a diferentes rutas, no recibe nigún props.

## Utilización
Este componente aparece en la página principal.
```js filename="index.tsx" {2}
<Grid.Col md={12} lg={12}>
    <BotonesAcciones />
</Grid.Col>
```

