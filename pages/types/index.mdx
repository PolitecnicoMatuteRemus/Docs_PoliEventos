# Types

En esta carpeta contiene los tipos o interfaces para los elementos de nuestra aplicación.
Se tienen dos documentos en esta carpeta las cuales una va a ser las interfaces para nuestra aplicación y la otra va a ser para la interfaz para nuestra base de datos.

El segundo archivo llamado **types_db.ts** es el que contiene los tipos para la base de datos, lo que serían los datos de las columnas. Y que campos se usan a la hora de algunos métodos de nuestra base de datos.
