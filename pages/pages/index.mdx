# Introducción - Pages Folder

https://nextjs.org/learn/basics/navigate-between-pages/pages-in-nextjs

En esta carpeta se tiene lo que sería las paginas o rutas que se muestran en la pagina. Tanto las configuraciones que tienen para que estas rutas se vean bien.

Los archivos mas importantes o los que siempre deben de estar son los siguientes:

    - _app.tsx
    - _document.tsx

Estos nos ayudan a configurar nuestra aplicación y colocar las propiedades para su uso.
